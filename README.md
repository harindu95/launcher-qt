# launcher-qt
An application launcher for linux

### Features

* Support Qute Browser
* Support terminal commands
* Support file searching 

### Requirements
* Qt5
* CMake
* Make

### Build

    git clone https://github.com/harindu95/launcher-qt
    cd launcher-qt
    mkdir build
    cd build 
    cmake ..
    ./launcher-qt
